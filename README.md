# WCF-Client-Library

Розроблена веб-служба WCF (проект WCF Service Application), яка за
надісланим значенням X повертає значення функції F4(X,3) з певної розробленої бібліотеки Library4. Окрім функції F4(), реалізувано ще дві функції - GetAllElements() і GetElementByID(int ID). Обидві функції повертають значення типу System.Data.DataTable. Перша функція повертає таблицю всіх елементів першої таблиці з бази даних "СалонКраси" (тільки версія MSSQL). Друга функція повертає тільки один рядок цієї таблиці, що містить тільки один елемент за заданим ID. 

Розроблена Windows-програму (проект Windows Forms Appication), додано
в неї посилання на веб-службу і при натисканні кнопки виводити значення функції F4(X,3), де аргумент X вводиться в текстовому полі (TextBox). Таблиці записів з бд "СалонКраси", які повертають функції GetAllElements() і GetElementByID(int ID), виводити при натисканні окремих відповідних кнопок за допомогою об'єктів DataGridView.
